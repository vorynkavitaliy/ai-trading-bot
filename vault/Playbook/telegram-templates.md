# Telegram-шаблоны для оператора

Сообщения отправляются в один чат (`TELEGRAM_CHAT_ID` из `.env`). Стиль строго на русском, без сленга. Шаблоны реализованы в `src/lib/tg-templates.ts` — менять там, не дублировать.

## Принципы

1. **Что произошло, почему, что дальше** — каждое сообщение содержит эти три части.
2. **Запрещённые слова:** «лонг», «фьюч», «шорт-сетап», «профит», «луп», «кэш-аут», «лонгуем», «шортуем».
3. **Разрешённые термины:** вход, выход, стоп, тейк, доход, убыток, размер, риск, регим (диапазон/тренд/переход), пара, аккаунт, ключ.
4. Направление сделки в латинице — **LONG** / **SHORT** (общепринятые ориентиры).
5. Все цифры с разделителем тысяч и фиксированным числом знаков после запятой.

## События

| Событие | Шаблон | Триггер |
|---|---|---|
| Открытие | `notifyOpen` | После успешного `execute.ts` хотя бы на одном аккаунте |
| Закрытие | `notifyClose` | SL hit / TP hit / abort / manual / external |
| Heartbeat | `notifyHeartbeat` | Раз в час (±10 мин), top-of-hour |
| Алерт | `notifyAlert` | Red-flag triggers (см. CLAUDE.md) |
| Отмена сетапа | `notifyAbort` | Снят pending limit без открытия позиции |

## Примеры

### Открытие

```
📈 ВХОД BTCUSDT LONG

Цена входа: 77 580.00
Стоп: 77 100.00 (риск 0.6%)
Тейк-1: 78 340.00 (50% объёма, затем стоп → безубыток)
Тейк-2: 79 200.00 (50% объёма)

Размер общий: 0.34 BTC
Аккаунты: 2 успешно
  • 200000/Vitalii qty=0.27
  • 50000/Ivan qty=0.07

Обоснование:
Регим — диапазон. Цена коснулась нижней границы Боллинджера на 1H, RSI 31, объём 1.6× от среднего за 20 баров. Кросс-биржевой OI растёт, но top-trader L/S 1.18 (умеренно длинный), что снижает риск squeeze. Funding −0.008% — давление на шорты.
```

### Закрытие по тейк-1

```
✅ ЗАКРЫТИЕ BTCUSDT LONG — тейк-1, затем стоп в безубыток

Цена входа: 77 580.00
Цена выхода: 78 340.00
Доход: $260 (+0.50R)
Комиссии: $4.20
```

### Закрытие по стопу

```
❌ ЗАКРЫТИЕ ETHUSDT SHORT — стоп

Цена входа: 2 312.50
Цена выхода: 2 327.00
Убыток: $-220 (-1.05R)
Комиссии: $2.10
```

### Heartbeat

```
✅ Жив. C0142.
Регим: BTC = диапазон, ETH = переход.
Открытых позиций: 1.
Доход дня: $+240 (+0.10%).
Триггеров в окне: 2/8.
```

### Алерт — мягкий стоп

```
⚠️ Дневной убыток ≥ −2.5% — мягкий стоп торговли

Дневной P&L: $-6 100 (-2.51%) на equity $243 692.

Что делать: новые входы заблокированы до 00:00 UTC. Открытые позиции остаются под управлением своих стопов.
```

### Алерт — расхождение

```
🛑 Расхождение vault ↔ Bybit

На Bybit найдена позиция BTCUSDT LONG 0.32 BTC (200000/Vitalii), но в vault такой записи нет.

Что делать: запустить `npm run reconcile` повторно через 1 цикл. Если расхождение сохранится — оператор разбирается вручную.
```

## Правила отправки

- Каждое сообщение **синхронно** отправляется через `send()` (telegraf). Если упало — пишем в лог, но не блокируем cycle.
- HTML-разметка разрешена (`<b>`, `<i>`, переводы строк).
- Длина сообщения ≤ 4096 символов (лимит Telegram). Обоснование автоматически обрезается до 1500 знаков.
